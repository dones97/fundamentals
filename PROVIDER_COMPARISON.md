## Migration Path

You can easily switch between providers!

### Recommended Path:
1. **Start with Groq** (quick setup, test the app)
2. **Try Ollama** (if you want unlimited usage)
3. **Upgrade to Claude** (if you need best quality for important decisions)

### Switching Providers:
1. Go to sidebar
2. Select different provider from dropdown
3. Enter API key (if needed)
4. Click "Connect Provider"
5. Continue analysis with new provider
